This extension for Visual Studio Code adds [CoCart](https://cocart.xyz) snippets and autocompletion of functions, classes and hooks.

## Features

Autocomplete for:

    CoCart version: 3.1

    Functions : 8
    Actions   : 12
    Filters   : 139
    Constants : 20
