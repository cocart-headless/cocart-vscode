This extension for Visual Studio Code adds [CoCart](https://cocartapi.com) snippets and autocompletion of functions, classes and hooks.

## Features

Autocomplete for:

    CoCart core version: 3.11
    CoCart Plus version: 1.2.0

    Functions : 13
    Actions   : 16
    Filters   : 173
    Constants : 29
